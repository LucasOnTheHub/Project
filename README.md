# Project.

File-based project manager with a 3D spatial interface — every project is a solar system.

## Quick start

```bash
git clone https://github.com/LucasOnTheHub/Project..git
cd Project.
npm install
npm run build
node dist/cli/index.js scan
```

## M0 — Foundations

- Vault reader (scan + watch)
- Front-matter parser (YAML)
- SQLite index with FTS5 full-text search
- CLI debug tool (scan, stats, search, node, reindex)
